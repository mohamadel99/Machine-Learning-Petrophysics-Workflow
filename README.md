# Machine-Learning-Petrophysics-Workflow
Python scripts for well log processing and machine learning-based petrophysical analysis.
# Well Log Machine Learning Analysis
This repository contains Python scripts developed for predicting and analyzing well log data using machine learning techniques.

## Overview
The project focuses on handling well log datasets and applying machine learning workflows for subsurface analysis and petrophysical interpretation.
The scripts include:
- Data loading and preprocessing
- Log data manipulation
- Machine learning model implementation

## Project Structure
app.py  
Main script used to run the workflow.

lis.py  
Handles well log data processing and utilities.

ML.py  
Contains machine learning models and analysis functions.

## Data
The dataset used in this project is private and therefore not included in this repository.
To run the scripts, users must provide their own well log dataset.

## Requirements
Python 3.x

Recommended libraries:
- pandas
- scikit-learn
